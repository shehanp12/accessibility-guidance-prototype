Define focus styles

For: Design

Everyone should be able to access all interactive components with a keyboard or a similar device. It must be obvious to them which element or link is the current focus position on the page. The browser default is generally not good enough.

Make sure that the focus is clearly visible. You can do this by adding something, like an outline or icon, or changing the colour of part of the component. The change should have a good colour contrast.

We recommend the focus style in the NHS.UK frontend library.
