Use headings correctly

Design
User research

Headings describe the content, and for some people they provide an effective method of navigating the page.

The size and style of text is not the deciding factor about heading level. The page structure is what's important.

Structure headings for accessibility

The question for each heading is whether it's a sub-section of the previous heading. If not, it should be at the same level as (or higher than) the previous section.
