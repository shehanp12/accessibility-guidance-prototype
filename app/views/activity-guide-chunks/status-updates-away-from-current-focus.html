Status updates away from current focus

Testing

Important changes in content during a dynamic state should be made aware to screen readers in a way that does not interrupt their journey.

Where a message that provides information to the user on the success or results of an action (i.e. updated search results, after successfully submitting a form) this information needs to be conveyed programmatically.

How to check status messages have been implemented:

  • Use a screen reader to conduct an action that will trigger a message or a change in context dynamically
  • Check whether the message is announced and describes the action accurately.
