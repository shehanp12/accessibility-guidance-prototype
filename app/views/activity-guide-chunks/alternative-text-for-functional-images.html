Use alternative text for functional images

For: Frontend, Testing

Some images are "functional". That means that they trigger an action. If people can't see the image, they need an alternative. You can use "alt‑text" or sometimes "visually-hidden".

W3C has information about how to deal with different kinds of functional image.

For some common NHS examples of functional images, see the close and search icons and the logo in the header component in the service manual. Another example is the action link.
