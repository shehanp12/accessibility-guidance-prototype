How to write alt-text for functional images

If the image has a text label that conveys its meaning, the alt-text of the image should be "null". The attribute should be present, but empty.

An example would be a link to a health app which includes a brand image and the name of the app. The app name says the same as the brand image so the image doesn't need explaining.

The alt text (or SVG title) of a functional image should say what the function is, rather than describing what it looks like. For example, "Play" for a video, not "triangle", and "search", not "magnifying glass".
