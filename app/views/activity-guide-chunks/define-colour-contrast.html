Define colour contrast

Text contrast

Using colours that contrast well makes it much easier for people to read content comfortably. Ensure the colour contrast between the text colour and background colour meets the following minimum ratio:

  • Small text (4.5:1) – text smaller than 24px or 19px if bold.
  • Large text (3:1) – text larger than 24px or 19px if bold.

Non-text contrast

User interface components (e.g. text inputs, radio buttons) and meaningful graphics need to be discernible by people with limited vision.

Graphics such as icons that are essential for understanding the content are required to have contrast. If an icon also has text, contrast is not required but it is recommended.

Inactive components are exempt from this requirement (e.g. a button that is disabled), although if there is a way of providing contrast without it being confusing, that would help people with low vision.
