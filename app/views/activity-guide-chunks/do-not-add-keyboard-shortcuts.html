Do not add keyboard shortcuts

Testing

Single letter keyboard shortcuts must not be implemented.

For example, Gmail uses the "j" and "k" keys to move between emails. Any keyboard shortcut using single numbers, punctuation, lowercase letters, uppercase letters or symbol characters can interfere with voice input software.

For 3rd party content the does implement these types of shortcuts: There must be either a mechanism which allows users to turn off character key shortcuts, or a functionality to remap the shortcuts to use more than one character (e.g. ALT + j).
