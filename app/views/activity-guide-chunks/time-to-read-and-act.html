Give users time to read and act

Testing

If you have set a time limit for inactivity (for example, on a multi-step form), it should not be less than 20 hours.

If you use 3rd party content that has a time-limit, it should warn users clearly how much time they have left and allow them to extend or turn off the time limit.

There are exceptions to this when the time limit is essential, for example, for an online test or real-time event.
