Data tables

Tables should only be used if there is a relationship between the "header" cells, and the "data" cells.

Assistive technology makes use of the relationship to announce the header before each cell and provide column and row information.

In the example data table, the last row would be read as: "Skin symptoms: Itchy blisters. Possible cause: shingles, chickenpox.", the headers and data are read out together.

It is important to only use tables where this type of relationship is required, otherwise it leads to confusing content.
