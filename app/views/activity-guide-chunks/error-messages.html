Make sure that error messages clearly describe what went wrong and how to fix the problem.

If you have an error summary at the top of a form, check that each item in the list has a link that moves the focus to the correct form field. This helps users who rely on keyboard navigation.

Include error messages next to each input. It should be visibily obvious that the message is associated with the input.

More about error messages and error summaries
Use the error message and error summary components in the service manual. They have already been tested for accessibility. They also contain links to useful GOV.UK guidance on writing good error messages.
