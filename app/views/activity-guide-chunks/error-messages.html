Error messages

Testing

Error messages helps users to identify invalid input and rectify the error, where invalidation occurs define error messages, these should be informative and clearly describes the issue.

Where an error list is used above the form, each item should be a link that moves the focus to the correct form field, this helps users who rely on keyboard navigation;

Error messages must be included next to each input, they should be clear, and they should be visibly associated with the input.
